# AI Team Leader Report Skill (v2.0) - Distribution

This package contains the fully optimized AI Team Leader Report generator skill for Gemini CLI.

## How to Install:
1. Extract the contents of `ai-team-leader-report-skill-v2.zip`.
2. Move the extracted folder into your project's `.agents/skills/` directory.
3. Ensure the `mes-pdamda-api` skill is also available, as this skill depends on it for data fetching.

## Features:
* **Interactive Dashboard:** 72-hour shift comparison slider.
* **Visual Analytics:** Stacked bar charts for output by part type and donut charts for loss breakdown.
* **Incident Log:** Automatically fetches and displays interruptions from MES.
* **AI Reasoning:** Identifies performance/quality deviations and suggests corrective actions.
* **Bosch Brand Aligned:** Follows official Bosch Sans and Gradient Banner guidelines.
